## Excel Dashboard Projects

## Overview

This repository contains dashboard and reporting projects developed using Microsoft Excel.

## Skills Demonstrated

* Pivot Tables
* Pivot Charts
* Conditional Formatting
* XLOOKUP
* Data Validation
* Dashboard Design

## Projects

### Personal Finance Dashboard

Tracks:

* Income
* Expenses
* Savings Goals
* Budget Categories

### Enrollment Dashboard

Tracks:

* Applications
* Enrollment Status
* Processing Metrics

### Sales Dashboard

Tracks:

* Revenue
* Transactions
* Monthly Trends

## Objectives

Develop business reporting skills while learning how to communicate data effectively through dashboards and visualizations.
